# Filson Pro fonts

Place the following woff2 files here. The CSS @font-face declarations in
`app/globals.css` are already wired up — the font will activate as soon as
the files are present.

Required files (weights 350 / 400 / 500 / 700 / 800):

- FilsonPro-Book.woff2
- FilsonPro-Regular.woff2
- FilsonPro-Medium.woff2
- FilsonPro-Bold.woff2
- FilsonPro-Heavy.woff2
